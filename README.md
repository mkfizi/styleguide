# Tailstart - Style Guide
[Tailstart](https://tailstart.github.io) style guide library..

## Installation
Clone this repository to get started.

Run the following command in project directory to install dependencies.
```bash
npm install
```

To compile the CSS file, use one of the following commands:
```bash
npm run build
npm run watch
```
